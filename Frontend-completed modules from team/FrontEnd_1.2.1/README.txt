This is a documentation for TechDocs implemented using Streamlit library

1. In order to run the application file which is streamlit.py, you need to have Python installed in your system along with libraries such as Streamlit.

2. To install Streamlit, use the below command

pip install streamlit

3. Once Streamlit is installed, import the library and start coding

4. When the code is done, if you want to run the application, Use the below commands

streamlit run <nameofyourfile.py>

In my case, it will be

streamlit run streamlit.py

This will give a URL, copy it and paste it in any browser, it'll show the application

5. As of now, the app is configured with Light Mode in browsers, Some aspects will not be properly visible if the browser is in dark mode.
