Help & FAQs

What is TechDocs ? +

TechDocs is a great on-line LaTeX editing tool that allows you to create LaTeX documents directly in your web browser. It is available as an on-premises or a cloud-based offering.

How does different users in different roles can achieve their goals using TechDocs ? +

TechDocs help different users achieve their goals as below:

1. Authors, Teachers, Publishers - Provides tools to write, edit, collaborate, submit, and manage articles through to publication

2. Groups - Provides simple functionality to share technical docs between peers

3. Students and Researchers - TechDocs professional features that provide streamlined branded enrollment and a resource portal with featured templates, training and administrative analytics and metrics to help students, researchers, faculty and staff in universities collaborate

What are the types of subscriptions available? +

There are two types of subscriptions available.

1. Techdocs Standard
2. Techdocs Premium

How to register with Techdocs? +

If you don’t already have an account, you need to register yourself to create an account. You can register by using your Google account or your Email address. If you already have an account, click Login in the upper right corner, then type in your email and password and click Login. After you are logged in, you should see the TechDocs Dashboard page.

How to reset the password? +

You can simply click on the Forgot Password link on the Login page and submit your email id. An auto-generated email containing a password reset link is sent to your email id. Using that you can reset your password seamlessly.

How to create a new document using Techdocs? +

Using Techdocs we can create anew document by the following steps:
 i) Log in to TechDocs by using your login credentials.
 ii) On the Dashboard page, click New Document --> Blank Project.
 iii) Enter a name for your project and click on the “tick mark” button to save the file.

How to edit a document using Techdocs? +

You can edit your LaTeX documents in TechDocs Dashboard by clicking on the file. Once you have made
the required changes, you can get back to doing other work as the Auto-Save feature takes care of saving
the document as you keep editing it making it a hassle-free save.

How to share a document using Techdocs? +

You can share your LaTeX documents with your peers in TechDocs Dashboard by clicking on the Action
button against a file. Update the user’s email id with whom you would like to share your document and
select the type of permissions you would like to provide. Click on Confirm. The file is visible to the other
user in the Shared by Others section on the Dashboard.

How to export the document using Techdocs? +

You can export TechDoc projects as a PDF file.
To download the PDF file
 i) Open the file from the Dashboard.
 ii) On the top right corner, click Download as PDF icon.

To download a LaTex file
 i) Go to Dashboard.
 ii) Select the file you wish to download.
 iii) Click on the Download button in the Actions section.

How to delete a document using Techdocs? +

To delete a file, you must select it by clicking on the checkbox as shown below and then hit the Delete icon on the right across the file. The file is moved to Trash from where you can either delete it permanently or  restore it.

How to search for a document using Techdocs? +

For searching, you can type a character or a string in the Search box and it starts listing all the files in your
Dashboard available for you.
